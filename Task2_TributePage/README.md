# Tribute Page - Ratan Tata

A modern and responsive Tribute Page dedicated to Ratan Tata, one of India's most respected business leaders and philanthropists.

This project is developed using HTML and CSS with beautiful UI design, animations, responsive layout, and attractive sections.

## Features

-  Modern Premium UI
-  Fully Responsive Design
-  Smooth Hover Animations
-  Attractive Hero Section
-  Life Journey Timeline
-  Achievement Section
-  Qualities & Legacy Cards
-  Beautiful Background Design
-  Smooth Scrolling Effect

## Technologies Used

- HTML5
- CSS3
